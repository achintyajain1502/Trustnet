# 🚀 TrustNet

TrustNet is a full-stack web application developed as part of a Project Based Learning (PBL) initiative. The project focuses on building a secure, scalable, and user-friendly digital platform that connects users with various services while providing administrators with complete control over system operations.

The system is designed with a clear separation between user and admin functionalities. Users can register, log in, and access available services through an intuitive interface, while administrators can monitor, manage, and update data through a dedicated admin dashboard.

TrustNet emphasizes real-world development practices such as frontend-backend integration, database management, authentication workflows, and deployment using modern tools. The project demonstrates the implementation of a complete web ecosystem, making it suitable for academic as well as practical learning purposes.

## 🔄 Workflow

1. User registers or logs in  
2. User accesses services  
3. Admin logs into dashboard  
4. Admin manages users and services  
5. System updates data in MongoDB
